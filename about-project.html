Watchlist Pro: Movie Manager

Purpose of the System

The purpose of Watchlist Pro is to provide movie enthusiasts with a simple, interactive, and persistent way to discover new movies and track what they want to watch. It solves the problem of forgetting movie recommendations by storing them in an easy-to-use digital dashboard.

System Features

  • Movie Discovery: Browse a list of popular movies and view detailed information via modal popups.
  • Dynamic Watchlist: Add movies directly from the browse page or input custom movie titles.
  • Progress Tracking: Mark movies as "Not Yet" or "Done" with visual color-coded indicators.

Technologies Used

  • HTML5: For structuring the web pages and creating forms.
  • CSS3: For styling, layout, and creating a responsive, modern user interface.
  • JavaScript: For DOM manipulation, form validation, event handling, and Local Storage integration.
